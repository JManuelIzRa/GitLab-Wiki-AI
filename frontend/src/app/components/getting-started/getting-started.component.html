Para conectar un repositorio necesitas tres cosas:

  • URL de GitLab — la dirección de tu instancia (ej. https://gitlab.com)
  • Ruta del proyecto — grupo/subgrupo/proyecto (sin la URL)
  • Personal Access Token (PAT) — un token con alcances read_api y read_repository
Cómo crear un PAT en GitLab
  1. Ve a Configuración → Tokens de acceso en tu perfil de GitLab
  2. Crea un token con alcances read_api y read_repository
  3. Copia el token (solo se muestra una vez)
  4. Pégalo en el campo «Personal Access Token» del formulario

El wiki se organiza con una barra lateral que lista todas las páginas generadas. Puedes navegar entre ellas de varias formas:

/ Buscar páginas en el sidebar
? Mostrar todos los atajos de teclado
Alt+← / Alt+→ Página anterior / siguiente
Cmd+K Abrir paleta de comandos
T Alternar tema claro/oscuro
Esc Cerrar panel o modal activo

Cada página tiene acciones disponibles: editar el contenido Markdown, regenerar con IA, y exportar la página.

El panel de preguntas (RAG) te permite conversar sobre el código del repositorio. Está disponible desde el botón flotante + en la esquina inferior derecha del wiki.

El sistema combina dos fuentes de contexto para responder:

  • Contexto del wiki — las páginas generadas con la documentación del proyecto
  • Contexto del código fuente — fragmentos relevantes del repositorio indexados semánticamente

Preguntas de ejemplo:

  • «¿Cuál es la arquitectura general del proyecto?»
  • «Explica el flujo de autenticación paso a paso»
  • «¿Qué dependencias externas usa el módulo principal?»
  • «¿Cómo se manejan los errores en las API routes?»

Atlas tiene muchas funciones más allá de la lectura de wikis. Estas son algunas de las más útiles:

  • Búsqueda de código (⌕) — busca fragmentos específicos en el código fuente del repositorio
  • Grafo de dependencias (⊞) — visualiza las relaciones entre módulos y paquetes
  • Historial (●) — consulta el estado y las ejecuciones anteriores de indexación
  • Configuración (⚙) — ajusta preferencias de idioma, tema y caché
  • Paleta de comandos (Cmd+K) — acceso rápido a todas las acciones desde el teclado
  • Chat multi-repositorio en grupos — conversación cruzada entre repositorios de un grupo indexado
  • Exportación — descarga páginas individuales en formato .md, .html o .pdf
